# Mavlen Labs — Web Presence

Professional website for Mavlen Labs. Live site: https://www.mavlenlabs.com/

Short description
- Mavlen Labs showcases services, expertise, and executed projects built with Next.js and modern frontend tooling.

Quick links
- Website: https://www.mavlenlabs.com/
- Source: this repository

Features
- Lightweight, SEO-friendly Next.js app (App Router)
- Reusable React components for services, projects, blog, and workflow
- Accessible, responsive layouts and animations

Tech stack
- Next.js (App Router)
- React
- CSS modules + global styles
